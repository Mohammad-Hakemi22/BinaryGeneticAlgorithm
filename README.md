# BinaryGeneticAlgorithm
Binary Genetic Algorithm
<br />Fitness function = `21.5 + x*sin(4*pi*x) + y*sin(20*pi*y)`
<br/> `-3 <= x <= 12.1 , 4.1 <= y <= 5.8`
<br/> Maximizing the fitness function
